Web Applications & Software

[re:boot] Web Applications & Software consitently captivate audiences in the bustling environment of the Web. Our team follows a few major tenants when developing:

  • Clarity:

    Clear interfaces donâ€™t need manuals. We avoid ambiguity by developing concise user pathways that pay close attention to both language and the visual hierarchy of elements.

  • Familiarity:

    Familiarity is about recognizing elements of an interface even if it's a user's first time interacting with it. Simple examples of universal recalls include the envelope icon signifying email, or a red-colored button signifying an action like deleting an element.

  • Consistency:

    Consistency goes hand-in-hand with familiarity. Our Web Applications build visual and functional narratives that allow users to develop pattern recognition, resulting in a seamless usage experience.

  • Responsiveness:

    In the context of Web development, 'responsiveness' usually means the ability to function fully and comfortably on any device. However, to us that goes without saying, so we repurposed 'responsiveness' to mean an application's ability to successfully interact with a user in a way that gives them a clear understanding of what's happening at any given moment.

  • Aesthetics:

    Eye sores aren't good for retaining an audience. Our team carefully compliments the functional core of an application with visual displays that draw users into an enjoyable expierence.

OptiEd

Web App, Optimization

Opti-Ed, originally aimed at Northwestern University Kellogg Executive MBA students, brings clarity and transparency to course scheduling. Courses are scheduled quickly and efficiently using an optimization algorithm designed to minimize overall preference loss.

OptiEd

OptiED is a custom application that served two purposes: visualization of course-related information, and generation of optimized course schedules from student specified preferences.

Intune Health

Web App, Sentiment Analysis

Intune Health allows users and their wellness managers (nutritionists, trainers, doctors, psychiatrists, psychologists) to track how they feel in a wellness journal, using text and emojis. Journal entries are translated to a sentiment score, providing information on a user's well-being.
